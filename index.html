My first from scratch responsive web app. A quiz game, used to test javascript and jquery skills. This quiz game, by user control will run through a set quiz questions one at a time. It will provide feedback to the user’s input while keeping track of the question number and score, the score will be graded at the end based on the answers submitted, and then allow the user to restart the quiz upon completion.
